# FAQ: Сервис CloudApp

## Авторизация

**В: Почему не работает авторизация?**
О: Проверьте:
1. Правильность email и пароля (регистр важен)
2. Не заблокирован ли аккаунт (5 неудачных попыток)
3. Подтверждён ли email (проверьте папку "Спам")
4. Не используется ли VPN из запрещённой страны

Если не помогает — сбросьте пароль через "Забыли пароль?".

**В: Забыл пароль. Как восстановить?**
О: На странице входа нажмите "Забыли пароль?", введите email. Ссылка на сброс придёт в течение 5 минут. Ссылка действительна 1 час.

**В: Не приходит email для подтверждения. Что делать?**
О: 1. Проверьте папку "Спам" 2. Добавьте noreply@cloudapp.io в контакты 3. Запросите повторно через Профиль → Безопасность.

## Биллинг

**В: Какие есть тарифы?**
О:
- **Free** — бесплатно, 100 запросов/месяц
- **Pro** — $9.99/мес, 10000 запросов/мес, приоритетная поддержка
- **Enterprise** — от $99/мес, безлимит, SLA 99.9%

**В: Как оплатить подписку?**
О: Visa/Mastercard, PayPal, криптовалюты (BTC, ETH, USDT). Для Enterprise — по счёту.

**В: Как отменить подписку?**
О: Профиль → Биллинг → "Отменить подписку". Действует до конца оплаченного периода.

## API

**В: Как получить API ключ?**
О: Профиль → API → "Создать ключ". Храните ключ в безопасном месте, он показывается только один раз.

**В: Какие лимиты на API?**
О: Free — 100/мес, Pro — 10000/мес, Enterprise — безлимит. Rate limit: 10 запросов/секунду.

**В: Ошибка 429 Too Many Requests. Почему?**
О: Превышен rate limit. Подождите или повысьте тариф. Можно настроить retry с exponential backoff.

## Аккаунт

**В: Как сменить email?**
О: Профиль → Настройки → "Сменить email". На новый адрес придёт ссылка для подтверждения.

**В: Как удалить аккаунт?**
О: Профиль → Настройки → "Удалить аккаунт". Данные удаляются в течение 30 дней (требование GDPR).

## Известные проблемы

**В: Медленно работает приложение**
О: Возможные причины:
1. Проблемы с интернетом
2. Пиковая нагрузка (проверьте status.cloudapp.io)
3. Устаревшая версия (обновите приложение)

**В: Ошибка 500 при загрузке файлов**
О: Известная проблема в версии 2.3.1. Исправлено в 2.3.2. Обновите приложение.

**В: Не синхронизируются данные**
О: 1. Проверьте интернет 2. Выйдите и войдите снова 3. Очистите кэш приложения.
